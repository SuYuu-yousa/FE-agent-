# 浏览器是什么





**解析、执行、渲染、存储、通信**

## 1. 浏览器打开一个网页时发生了什么

流程很简单：

1. 浏览器发请求
2. 服务器返回 HTML
3. 返回内容里分两部分理解：
   - **响应头**：先告诉浏览器“这页面该怎么处理”
   - **响应体**：真正的 HTML 内容

比如服务器会先说：

- 这个页面是什么编码
- 能不能执行外部脚本
- 能不能加载第三方资源
- 要不要缓存

然后浏览器再去解析 HTML。
