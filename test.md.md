

# K-means 聚类算法

## (a) 算法流程及伪代码

### 流程

1. 随机选取 K 个点作为初始聚类中心
2. 计算每个样本到各中心的距离，将其分配到最近的簇
3. 更新每个簇的中心为簇内所有点的均值
4. 重复 2-3，直到中心不再变化或达到最大迭代次数

```
8.     end for
9. until 聚类中心不再变化 or 达到最大迭代次数
```

## (b) 能否得到全局最优解？

**不能。** K-means 的目标函数 J=∑j=1K∑xi∈Cj∥xi−μj∥2J=∑j=1K​∑xi​∈Cj​​∥xi​−μj​∥2 是非凸的，算法采用贪心迭代策略，对初始中心的选择敏感，只能收敛到局部最优解而非全局最优。

## (c) 无监督学习与有监督学习的区别

- **有监督学习**：训练数据带有标签 (xi,yi)(xi​,yi​)，目标是学习输入到输出的映射，如分类、回归。
- **无监督学习**：训练数据无标签，目标是发现数据的内在结构和分布规律，如聚类、降维。

**核心区别**：有无标签信息的指导。
